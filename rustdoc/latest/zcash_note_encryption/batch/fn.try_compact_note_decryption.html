Trial decryption of a batch of notes for light clients with a set of recipients.

This is the batched version of crate::try_compact_note_decryption.

Returns a vector containing the decrypted result for each output, with the same length and in the same order as the outputs were provided, along with the index in the ivks slice associated with the IVK that successfully decrypted the output.
